SpotJournal

SpotJournal is a web application where users write journal entries about their emotions, comparing their emotional state before and after listening to a random, familiar song based on their personal Spotify data. I worked on the back-end where I learned how to create API requests and handle responses, specifically with Spotify's Web API and Youtube's Data API. Then, we stored relevant data entries using mySQL with Flask as our framework, and deployed it on Google Cloud Platform.

Devpost Submission for HackDuke 2020

GitHub


Languages & Technologies: Python, Flask, mySQL, Google Cloud Platform

Movie Classifier

In this project, I built and tested a supervised machine learning k-nearest-neighbors classifier that could guess whether a movie is a comedy or a thriller, using only the number of times words appear in the movies's screenplay. I learned how classification works and the statistical analysis behind it as well, such as the application of linear regression and classification onto categorical data.

Languages & Technologies: Python, NumPy, Matplotlib, Jupyter
